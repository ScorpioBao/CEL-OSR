## Causal Evidence Learning for Trusted Open Set Recognition under Covariate Shift
![本地路径](CEL-OSR.png)
## Installation
```
conda create --name cel python==3.8
conda activate cel
conda install pip
git clone https://github.com/ScorpioBao/CEL-OSR.git
cd CEl-OSR
pip install -r requirements.txt
```
